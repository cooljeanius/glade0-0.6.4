Opzioni principali

Directory del progetto

La directory in cui si trovano i progetti Glade. Potete sceglierla usando una finestra di dialogo per la selezione di file, richiamabile cliccando su Sfoglia.

Nome del progetto

Il nome del vostro progetto. Modificandolo cambieranno il nome del progetto e quello del file contenente il progetto.

Nome dell'applicazione

Il nome dell'eseguibile che verrà generato da Glade per il vostro progetto. Modificandolo il nome del file del progetto cambierà automaticamente.

File del progetto

Il nome del file del progetto Glade. E' il file in cui verranno memorizzate tutte le informazioni riguardanti il vostro progetto.

Sottodirectory

Directory dei sorgenti

La directory in cui Glade metterà il suo codice sorgente (quando gli verrà detto di farlo).

Directory delle immagini

La directory in cui Glade cercherà tutte le immagini usate nel progetto.

Linguaggio

Selezionate il linguaggio di programmazione in cui Glade scriverà il codice. Al momento della scrittura di questo documento le opzioni possibili sono: C, C++, Ada95, Perl ed Eiffel.

GNOME
