Opzioni generali

Supporto per Gettext

Glade deve scrivere codice che supporti lo GNU gettext per l'internazionalizzazione?

Imposta nomi dei widget

Glade deve scrivere nel codice i nomi di tutti i widget? Questa opzione è utile se avete intenzione di usare le chiamate Gtk+ come gtk_widget_path(). Vedete anche gtk_widget_set_name().

Crea copie di backup dei file sorgente

Glade deve creare delle copie di backup di tutti i vostri file sorgente quando ne create di nuovi?

Opzioni File in uscita

Crea file main.c

Glade deve creare un file main.c contenente un semplice codice d'inizializzazione? Se avete abilitato il supporto per GNOME, allora chiamerà gnome_init(), altimenti chiamerà gtk_init().

Crea Funzioni di supporto

Glade deve creare il file support.c, contenente delle funzioni di servizio per compiti quali la creazione di immagini, o la ricerca di un widget noto il nome?

Crea file di supporto per la compilazione

Glade deve creare dei file utili per supportare la configurazione del vostro progetto tramite automake/autoconf? Abilitando questa opzione verranno creati diversi file:

AUTHORS: File per registrare i nomi dei collaboratori.
ChangeLog: File con il log delle modifiche da voi fatte al progetto.
Makefile.am: template del progetto per Automake.
NEWS: Annunci delle nuove versioni rilasciate al pubblico.
README: Informazioni generiche sul progetto.
acconfig.h: Definisce i simboli predefiniti del preprocessore per autoconf.
autogen.sh: Script per la creazione di uno script di configurazione.
configure.in: Template per la scrittura di uno script configure.
macros/: Directory contenente le macro di automake per Glade, GNOME, etc.
stamp-h.in: File Timestamp.
src/Makefile.am: Template di automake per la directory src/.
